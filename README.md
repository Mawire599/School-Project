MegAsolutionsAssignment is a C# console-based payroll application developed to demonstrate core programming concepts such as input handling, calculations, and error management.

This project is part of my Software Development learning journey and showcases my ability to build simple business logic applications.

Features:
• Capture employee details (Name, Hours Worked, Hourly Rate).
• Calculate total salary automatically.
• Generate a simple payslip output.
• Input validation for correct data entry.
• Basic error handling (format exceptions)

Technologies Used:
• C#
• .NET Console Application
• Visual Studio / VS Code

Project Structure:
