# MegAsolutionsAssignment

A C# Windows Forms school project that generates a simple payslip.

## What it does

- Accepts employee name, salary, and hours worked
- Calculates gross pay, tax, and net pay
- Displays a formatted payslip in the application

## Technologies

- C#
- .NET (Windows Forms)
- Visual Studio / Visual Studio Code

## How to run

1. Open `MegAsolutionsAssignment.sln` in Visual Studio.
2. Build the solution.
3. Run the application.

## Usage

1. Enter the employee full name.
2. Enter the hourly salary.
3. Enter the hours worked.
4. Click `Generate Payslip`.
5. Review the payslip output in the right-side panel.
